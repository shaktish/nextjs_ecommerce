# Project setup

## Create Next App
Creating Next App
- will scaffold a new Next.js application inside a folder named client
- `npx create-next-app@latest client`

## Install UI Library - shadcn/ui 
### For this project we are using shadcn/ui - its a component library 
- open source built on top of react 
- less opinionated 
- Potentially lighter if you cherry-pick components and styles
Run the below command
- it's used to initialize the shadcn/ui component library in your React or Next.js project.
- Without running init, the CLI cannot scaffold components because it doesn’t know your project structure or Tailwind setup.
- It’s the first step to start using shadcn/ui in your project. 
``` npx shadcn@latest init ```

### Lets insall components 
```sh
npx shadcn@latest add button card input select textarea dropdown-menu sonner label badge checkbox dialog sheet
```
